How It All Began

My fascination with technology started at age 13, when I created my first local webpage using HTML and CSS. I was amazed when I saw the page on Google and discovered that I could modify everything with code. What began as a hobby and curiosity became a passion when I built my first web application at 16 using JavaScript.

The thrill of solving problems through code motivated me to pursue computer science professionally, leading me to where I am today.
